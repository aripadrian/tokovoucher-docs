# Persiapan Awal

- Dapatkan `Secret Key` & `Member Code` di halaman [Pengaturan Secret Key ](https://member.tokovoucher.id/pengaturan/secret-key)
- Silahkan set `Content-Type` pada header menjadi `application/json` jika menggunakan method POST
- Seluruh transaksi via API akan menggunakan method `GET` & `POST`


> Semua request pada API menggunakan URL : <br />
> `https://api.tokovoucher.id/` untuk jalur API method POST dan GET
> `http://trx-ip.tokovoucher.id/` untuk jalur IP
