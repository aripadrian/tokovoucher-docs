Menampilkan daftar harga

### Endpoint
<div style="font-family:monospace;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
<span style="margin-left:4px;color:#e74c3c;background:#fed1cc;border-radius: 4px;padding: 4px;">https://v1.tokovoucher.net/member/produk/list/[member_code]/[signature]</span>

</div>

### Deskripsi
Berikut ini parameter / payload yang di harapkan :

| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| member_code | Member code anda                                                                                                       | String    | Ya    |
| signature   | Signature dengan formula **md5(ref_id:member_code:kode_produk:tujuan:secret)**    Note: Secret bisa dilihat pada member area | String    | Ya    |
