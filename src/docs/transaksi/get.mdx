### Endpoint
<div style="font-family:monospace;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
<span style="margin-left:4px;color:#e74c3c;background:#fed1cc;border-radius: 4px;padding: 4px;">https://v1.tokovoucher.net/v1/transaksi?ref_id=ref342212x42effdde3333&produk=FFxx&tujuan=034344234&signature=911094d45865971272ad61742feeaf37&member_code=MEMBER20220801UAXH95&server_id=2343</span>

</div>

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| ref_id      | Ref ID unik anda                                                                                                       | String    | Ya    |
| produk      | Kode produk                                                                                                            | String    | Ya    |
| tujuan      | Tujuan pengisian                                                                                                       | String    | Ya    |
| server_id   | Isi string kosong "" jika produk tidak memiliki Server ID                                                              | String    | Ya    |
| member_code | Member code anda                                                                                                       | String    | Ya    |
| signature   | Signature dengan formula **md5(ref_id:member_code:kode_produk:tujuan:secret)**    Note: Secret bisa dilihat pada member area | String    | Ya    |
