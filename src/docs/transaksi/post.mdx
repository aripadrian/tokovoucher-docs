Melakukan pembelian produk menggunakan Metode POST

### Endpoint
<div style="font-family:monospace;margin-bottom: 12px;">
<span style="color:white;background:#27ae60;border-radius: 4px;padding: 4px;">POST</span>
</div>

```
https://api.tokovoucher.id/v2/transaksi
```

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| ref_id      | Ref ID unik anda                                                                                                       | String    | Ya    |
| produk      | Kode produk                                                                                                            | String    | Ya    |
| tujuan      | Tujuan pengisian, bisa menggunakan Player ID atau User ID                                                                                                     | String    | Ya    |
| server_id   | Isi string kosong "" jika produk tidak memiliki Server ID                                                              | String    | Ya    |
| member_code | Member code anda                                                                                                       | String    | Ya    |
| Secret   | Note: Secret bisa dilihat pada member area | String    | Ya    |

```json title=Contoh
{
    "ref_id":"tx334234",
    "produk":"FF5",
    "tujuan":"034344234",
    "server_id":"exr33f"
    "member_code":"MEM3532234432X34",
    "secret":"f109a98c4f298ade23fb6c1f067dce53"
}
```
