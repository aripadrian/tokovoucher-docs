Melakukan pembelian produk menggunakan Metode GET dan response text untuk software Otomax, IRS dan lain lain

### Endpoint
<div style="font-family:monospace;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
<span style="margin-left:4px;color:#e74c3c;background:#fed1cc;border-radius: 4px;padding: 4px;">https://api.tokovoucher.id/v1/transaksi?ref_id=[ref_id]&produk=[kode_produk]&tujuan=[tujuan]&secret=[secret]&member_code=[member_code]&server_id=[server_id]</span>

</div>

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| ref_id      | Ref ID unik anda                                                                                                       | String    | Ya    |
| produk      | Kode produk                                                                                                            | String    | Ya    |
| tujuan      | Tujuan pengisian                                                                                                       | String    | Ya    |
| server_id   | Isi string kosong "" jika produk tidak memiliki Server ID                                                              | String    | Ya    |
| member_code | Member code anda                                                                                                       | String    | Ya    |
| secret   | Secret Key Anda Note: Secret bisa dilihat pada member area | String    | Ya    |
