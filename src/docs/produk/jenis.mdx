Menampilkan operator produk yang tersedia

### Endpoint
<div style="font-family:monospace;margin-bottom:12px;">
<span style="color:white;background:#e74c3c;border-radius: 4px;padding: 4px;">GET</span>
</div>

```
https://api.tokovoucher.id/member/produk/jenis/list/[member_code]/[signature]/[id_operator]
```

### Deskripsi
Berikut ini parameter / payload yang di harapkan :


| Parameter   | Deskripsi                                                                                                              | Tipe Data | Wajib |
|-------------|------------------------------------------------------------------------------------------------------------------------|-----------|-------|
| member_code | Member code anda                                                                                                       | String    | Ya    |
| id_operator | Operator Produk                                                                                                       | String    | Ya    |
| signature   | Signature default anda Note: Signature detault bisa dilihat pada member area | String    | Ya    |
